# WEBGL Kabaa Project

This project simulates [The Holy Kabaa](https://en.wikipedia.org/wiki/Kaaba) using a particles system. Built using `THREE.js`, and `GPGPU`.

## To run locally

```bash
yarn && yarn start
```

Then visit http://localhost:8080
